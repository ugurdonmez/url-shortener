Tech stack

  • Scala
  • Play Framework
  • Redis
  • ScalaTest
  • Mockito
  • Docker

Commands

Run

Run docker-compose, it will start api, redis and will expose api port to host.

docker-compose up

Test

sbt test

Sample usage


            # Shorten
            curl -X POST http://localhost:9000
            # Call shortened url
            curl -X GET http://localhost:9000/igDpdNx
